# **App Name**: Personal Portfolio

## Core Features:

- Hero Section: A welcoming hero section with a brief, engaging greeting.
- Floating Navbar: Sticky navigation bar with links to About Me, Projects, and Connect (LinkedIn & GitHub).
- Tech Stack Ticker: A dynamic, running ticker displaying mastered technologies.
- About Me Section: Detailed section introducing the portfolio owner.
- Projects Showcase: Dedicated space to present personal projects with descriptions and links.
- Connect Section: Direct links to LinkedIn and GitHub profiles for professional networking.

## Style Guidelines:

- Background color: Dark grey (#222831) for a modern feel.
- Component color: Light grey (#948979) to provide contrast.
- Text color: Dark grayish-blue (#393E46) for readability.
- Font: 'Geist' (sans-serif) for general text, and 'Geist Mono' (monospace) for code snippets or monospaced elements. Note: currently only Google Fonts are supported.
- Simple and straightforward layout with consistent spacing and padding.
- Component radius: Rounded corners with a 25px radius for a soft look.
